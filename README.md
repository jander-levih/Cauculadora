# 🧮 Calculadora Científica Completa

Uma calculadora web completa com funcionalidades científicas, cálculos de data e proporcional, desenvolvida em HTML, CSS e JavaScript.

## ✨ Funcionalidades

### 🔬 Calculadora Científica
- **Operações básicas**: Adição, subtração, multiplicação, divisão
- **Funções trigonométricas**: Seno, cosseno, tangente
- **Funções logarítmicas**: Logaritmo base 10, logaritmo natural
- **Funções matemáticas**: Raiz quadrada, potência ao quadrado, fatorial
- **Constantes**: π (pi), e (número de Euler)
- **Outras funções**: Valor absoluto, inverso, porcentagem
- **Teclas de atalho**: Suporte completo ao teclado

### 📅 Calculadora de Datas
- **Cálculo de períodos**: Diferença entre duas datas
- **Adicionar/Subtrair dias**: Calcular datas futuras ou passadas
- **Interface intuitiva**: Seletores de data nativos

### 💰 Calculadora Proporcional
- **Cálculo proporcional**: Valor devido baseado no tempo de uso
- **Exemplo prático**: 
  - Produto: R$ 119,00
  - Período total: 30 dias
  - Período usado: 18 dias
  - Valor devido: R$ 71,40
- **Histórico completo**: Salva todos os cálculos realizados
- **Persistência local**: Histórico salvo no navegador

## 🚀 Como Usar

### Calculadora Científica
1. Use os botões numéricos para inserir números
2. Selecione a operação desejada
3. Para funções especiais (sin, cos, log, etc.), digite o número primeiro
4. Use `C` para limpar tudo, `CE` para limpar entrada atual
5. Use `⌫` para apagar o último dígito

**Teclas de atalho:**
- `0-9`: Números
- `+`, `-`, `*`, `/`: Operações
- `Enter` ou `=`: Calcular resultado
- `Escape`: Limpar tudo
- `Backspace`: Apagar último dígito

### Calculadora de Datas
1. **Cálculo de Períodos:**
   - Selecione data inicial e final
   - Clique em "Calcular Diferença"
   
2. **Adicionar/Subtrair Dias:**
   - Selecione uma data base
   - Digite o número de dias
   - Clique em "Adicionar Dias" ou "Subtrair Dias"

### Calculadora Proporcional
1. Digite o valor total do produto/serviço
2. Informe o período total (em dias)
3. Informe o período utilizado (em dias)
4. Clique em "Calcular Valor Devido"
5. O resultado será exibido e salvo no histórico
